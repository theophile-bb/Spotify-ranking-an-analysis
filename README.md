## Getting the data

Dataset : https://www.kaggle.com/datasets/yelexa/spotify200

This dataset includes:

- Track name

- Artist

- Weekly ranking

- Streams

- Release information

- Country and region metadata

- Music features (e.g., danceability, tempo, valence)
